معمارية النظام

ما هو المشروع؟

هذا المشروع عبارة عن سيارة روبوتية مصغّرة ذكية تتحكم فيها من هاتفك الذكي عبر البلوتوث. السيارة لا تنتظر أوامرك فقط — بل هي أيضاً تُفكّر بنفسها: إذا اقتربت من عائق، تتوقف فوراً وتُضيء مؤشر تحذير، ثم تعود لوضع الانتظار بعد إزالة العائق.

الهدف هو دمج ثلاث تقنيات أساسية في نظام واحد متكامل:

  1. التحكم اللاسلكي — التواصل مع الهاتف عبر Bluetooth
  2. قيادة المحركات — التحكم بـ 4 محركات وServo بدقة PWM
  3. الإحساس بالبيئة — قياس المسافة واكتشاف العوائق تلقائياً

كيف يعمل النظام؟

تسلسل العمليات الذي يحدث في كل لحظة:

1

⚡ التشغيل الأولي

عند تشغيل البطارية، الـ Servo يبدأ المسح من 90° (الأمام)، ووحدة Bluetooth تنتظر الأوامر من الهاتف.

2

🔄 المسح المستمر للبيئة

الـ Servo يتأرجح بين 20° و160° بسلاسة. عند كل زاوية (كل 30ms)، يقرأ المستشعر HC-SR04 المسافة الحالية باستخدام millis() لعدم تجميد البرنامج.

3

📲 استقبال أوامر الحركة

إذا لم يكن هناك عائق قريب ← السيارة تُنفّذ أمر الحركة أو السرعة القادم من الهاتف الذكي عبر UART.

4

⚠️ اكتشاف العائق الطارئ

مسافة أقل من 20 سم؟ توقف فوري + LED يضيء + البرنامج يتجاهل أي أوامر حركة إضافية حتى يُزال العائق.

5

✅ الاستئناف الذاتي

بعد إزالة العائق (مسافة > 20 سم) ← LED يُطفأ والسيارة ترجع لوضع الانتظار تلقائياً دون أي تدخل بشري.

أوامر التحكم عبر Bluetooth

الهاتف يرسل حرفاً واحداً. الأرقام 0–9 تتحكم في السرعة (0 = أبطأ، 9 = أسرع):

شرح الكود البرمجي

📏 1. قياس المسافة من HC-SR04

المستشعر يُرسل نبضة صوتية ويحسب وقت الصدى. الصيغة الرياضية:

المسافة = (الزمن × 0.034) ÷ 2
0.034 = سرعة الصوت (سم/ميكروثانية)
÷ 2 لأن الصوت يقطع المسافة ذهاباً وإياباً
getDistance.ino 
long getDistance() {
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH);      // أرسل نبضة 10µs
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);

  long duration = pulseIn(ECHO_PIN, HIGH, 30000);
  return duration * 0.034 / 2;        // حوّل الزمن إلى cm
}

🔄 2. المسح التفاعلي بدون delay()

بدلاً من delay() التي تُجمّد البرنامج، نستخدم millis() لتعدد المهام:

sweepServo.ino 
void sweepServo() {
  if (millis() - lastSweepTime >= 30) {
    lastSweepTime = millis();
    servoAngle += servoDir * 2;            // تحرّك 2 درجة

    if (servoAngle >= 160 || servoAngle <= 20)
      servoDir *= -1;                         // اعكس الاتجاه عند الحد

    myServo.write(servoAngle);

    long dist = getDistance();
    if (dist > 0 && dist < 20) {          // عائق! ⚠️
      obstacle = true;
      Stop();
      digitalWrite(LED, HIGH);
    } else {
      obstacle = false;
      digitalWrite(LED, LOW);
    }
  }
}

⚡ 3. الحلقة الرئيسية loop()

loop.ino 
void loop() {
  sweepServo();                              // مسح مستمر بدون توقف

  if (Serial.available() > 0) {
    command = Serial.read();                // استقبل أمر من Bluetooth

    if (obstacle) return;                   // عائق؟ تجاهل الأوامر

    switch (command) {
      case 'F': forward();       break;
      case 'B': back();          break;
      case 'L': left();          break;
      case 'R': right();         break;
      case 'S': Stop();          break;
      case '6': Speed = 204;    break;   // سرعة متوسطة
      case 'q': Speed = 255;    break;   // أقصى سرعة
    }
  }
}

🎮 4. دوال التحكم في المحركات

analogWrite() ترسل قيمة PWM (0–255) لضبط سرعة المحرك بدقة:

motors.ino 
void forward() {
  analogWrite(in1, Speed); analogWrite(in2, 0);
  analogWrite(in3, Speed); analogWrite(in4, 0);
}
void back() {
  analogWrite(in1, 0); analogWrite(in2, Speed);
  analogWrite(in3, 0); analogWrite(in4, Speed);
}
void left() {
  analogWrite(in1, 0); analogWrite(in2, Speed);
  analogWrite(in3, Speed); analogWrite(in4, 0);
}
void right() {
  analogWrite(in1, Speed); analogWrite(in2, 0);
  analogWrite(in3, 0); analogWrite(in4, Speed);
}
void Stop() {
  analogWrite(in1,0); analogWrite(in2,0);
  analogWrite(in3,0); analogWrite(in4,0);
}
